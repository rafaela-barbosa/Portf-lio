    About Me

    Hi! My name is Rafaela Barbosa, a Software Engineering student and an enthusiast of new technologies. I am still in development — just like the projects I share here! This portfolio is part of an academic activity, where I explore HTML, CSS and JavaScript to build a solid foundation in web development.

    Main Objective

    My goal is to transform every learning experience, inside and outside the classroom, into something practical and shareable. By documenting each mini-project on GitHub, I want to create a space to showcase my progress and explore the impact of my work. I enjoy building experiences on the front-end, but my true passion lies in the back-end. I'm currently delving into Python and Node.js.

    To-Do-List

    The "To-Do List" project aims to put into practice the knowledge acquired in JavaScript, CSS and HTML. Developed as part of an academic exercise, this project aims to create a simple and functional application, allowing the user to add, mark as completed and remove tasks from an interactive list.

    Website for a Celebrity

    The goal of this project was to improve the knowledge acquired in HTML and CSS in the creation of static pages. Develop a personal website for a celebrity, with a simple and efficient structure, providing a visually appealing and intuitive experience to fans and visitors.

    Informative Website

    The goal of this project was to explore different HTML and CSS configurations and techniques, with a focus on creating an informative static website. The website was developed to provide educational content on a specific topic, using a well-organized and easy-to-navigate structure.
